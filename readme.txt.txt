ADVISTA AI
Smart Billboard Marketplace with AI Analytics
==================================================

PROJECT OVERVIEW
--------------------------------------------------
AdVista AI is an AdTech platform that connects
billboard owners with advertisers through an
AI-powered marketplace.

It helps advertisers:
- Discover available billboards
- View demand hotspots on maps
- Get billboard recommendations for products
- Track campaign performance and ROI

It helps billboard owners:
- List ad spaces
- Manage bookings
- Showcase audience reach


FEATURES
--------------------------------------------------
• Billboard booking marketplace

• Product-based hotspot search using Google Maps

• AI billboard recommendation engine

• Campaign analytics dashboard
  - Impressions
  - Peak timing
  - ROI insights

• Payment workflow
  - UPI
  - Debit/Credit Cards
  - Net Banking / EMI


AI MODULES
--------------------------------------------------

• Product demand hotspot engine

• Billboard recommendation scoring model
